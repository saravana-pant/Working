# Saravanan A — Portfolio (static)

This is a simple static portfolio site (HTML, CSS, JavaScript).

Preview locally:

Using VS Code Live Server extension: open `site/index.html` and click "Go Live".

Or using Python (from the `site/` folder):

```powershell
# Python 3
python -m http.server 5173
# then open http://localhost:5173 in your browser
```

Files:
- `index.html` — main page
- `styles.css` — styles
- `script.js` — JS interactions
